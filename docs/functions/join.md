# join()

Joins an array to one string.

### Parameters

| Name | Type | Description
| ---- | ---- | -----------
| values | string[] | The values to join
| using | string | (Optional) The separator

### Example

Documentation source:

```
{{ join(values: ['a', 'b', 'c'], using: '|') }}
```

Result:

```
a|b|c
```
